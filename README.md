# arya-STARK

arya-STARK is a zk-STARK framework designed to secure Federated Learning
using provable computation and post-quantum cryptography.

## Features
- zk-STARK proofs for gradient correctness
- Post-quantum signatures (Dilithium)
- Byzantine-resilient aggregation support

## Disclaimer
This project is a research prototype and not production-ready.
